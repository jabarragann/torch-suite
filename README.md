# Pytorch Suite

Features included in torchsuite
* Checkpoints: Save models and training plots.
* Optuna: Hyperparameter tunning.
* Training loops: Pytorch training loops.


## Install

```
pip install -e .
pip install -e ./pytorch-checkpoint/
```
